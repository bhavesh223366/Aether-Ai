# 🌌 Aether AI Studio

**The Next Generation of AI Content Production.**

Aether AI Studio is a powerful, autonomous video production pipeline designed to turn simple text prompts into viral-ready short-form content. Powered by advanced **RAG (Retrieval-Augmented Generation)** and state-of-the-art AI models, Aether handles the entire creative process—from deep research and scriptwriting to voice synthesis and visual assembly.

---

## ✨ Why Aether AI Matters (USP)

In a world flooded with generic AI content, **Aether AI stands apart through factual grounding and creator-centric design.**

- **🧠 RAG-Driven Intelligence**: Unlike standard generators that often hallucinate, Aether's RAG Agent researches your topic in real-time across global knowledge bases (like Wikipedia) to ensure scripts are accurate, informative, and engaging.
- **⚡ Zero-to-Viral in Minutes**: Aether automates the most time-consuming parts of content creation. What takes a human editor 4 hours, Aether does in 4 minutes.
- **🎭 Multi-Dimensional Personalization**: Custom language support (9+ languages), emotional tone selection, and hyper-realistic AI voices (ElevenLabs) allow you to tailor content for any audience globally.
- **🚀 High-Retention Visuals**: Intelligent keyword extraction and automated image sourcing (Pexels/Pixabay) ensure every scene is visually stimulating and perfectly synced with the narration.

---

## 🛠️ High-Level Architecture

Aether AI is built with a modern, decoupled architecture for maximum performance and scalability.

- **Frontend**: Next.js 15+ with a "Cosmic Neon" aesthetic, featuring a robust Dashboard and real-time Scene Editor.
- **Backend**: Express.js consolidated API engine managing the AI production pipeline and secure credit management.
- **Database**: PostgreSQL (Prisma) with real-time persistence and user-specific content libraries.
- **AI Stack**:
  - **LLMs**: Groq (Llama 3.1) & OpenRouter (GPT-4o mini)
  - **Synthesis**: ElevenLabs Hyper-Realistic Voice AI
  - **Research**: Custom RAG Agent with Wikipedia Integration
  - **Visuals**: Pexels, Pixabay, and Gemini-powered Thumbnail Generation

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 20+
- PostgreSQL Database
- API Keys: Clerk, Groq/OpenRouter, ElevenLabs, Pexels

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/bhavesh223366/Aether-Ai.git

# Install dependencies for both frontend and backend
cd aether-ai/frontend && npm install
cd ../backend && npm install
```

### 3. Environment Setup
Create a `.env` file in both `frontend` and `backend` directories using the provided examples.

### 4. Run Development Servers
```bash
# In separate terminals
cd frontend && npm run dev
cd backend && npm run dev
```
