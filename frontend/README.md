# Aether AI Studio 🌌✨

Aether AI Studio is the next generation of AI-powered content production. Turn your ideas into viral, high-fidelity short videos for YouTube, TikTok, and Instagram in seconds.

## 🚀 Elevated AI Production
- **Aether Scripting**: Specialized RAG-enabled LLM logic for high-retention storytelling.
- **Elite Vocalist**: Hyper-realistic voiceovers powered by industry-leading TTS models.
- **Persona Studio**: Animate professional avatars or your own photos into high-fidelity talking heads.
- **Aero Navigation**: A premium, high-performance studio interface designed for elite creators.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Authentication**: Clerk
- **AI Engine**: Google Gemini (RAG), Groq (LLM), ElevenLabs (TTS), D-ID (Avatars)
- **Database**: Prisma + PostgreSQL
- **Aesthetic**: Vanilla CSS with premium Glassmorphism

## ⚡ Quick Start
```bash
npm install
npm run dev
```
